\section{Summary of Contributions}
\label{sec:concl-summary}

\textit{Recapitulate the main findings, one paragraph per contribution:}

\begin{enumerate}
    \item \textbf{NAVSIM City Splits}: First per-city benchmark for NAVSIM, enabling systematic cross-city generalization studies. Released as a community resource.

    \item \textbf{Cross-City SSL Generalization}: Demonstrated that SSL backbones (I-JEPA, DINOv2, MAE) reduce cross-city generalization gaps by an order of magnitude compared to supervised ResNet, across both TransFuser and LAW architectures. The finding that cross-city failure is a representation problem, not a planning architecture problem, has implications for how the community approaches robustness.

    \item \textbf{JEPA-Diffusion Planner}: Introduced a camera-only planner combining frozen I-JEPA features with truncated diffusion, achieving competitive EPDMS with real-time inference. First camera-only SSL method to report EPDMS on NAVSIM.

    \item \textbf{Planning Head Analysis}: Compared generative (diffusion) vs.\ regression planning heads under cross-city transfer, providing guidance on architecture selection for deployment across geographies.
\end{enumerate}

\section{Limitations}
\label{sec:concl-limitations}

\textit{Discuss honestly:}
\begin{itemize}
    \item Open-loop evaluation only (PDMS/EPDMS); closed-loop simulation would strengthen claims.
    \item Limited to 4 cities in nuPlan; broader geographic diversity (e.g., non-US/Singapore cities) is needed.
    \item SSL backbones pre-trained on ImageNet, not driving data; domain-specific pre-training may further improve results.
    \item Camera-only planner sacrifices LiDAR information; not suitable for all deployment scenarios.
\end{itemize}

\section{Future Work}
\label{sec:concl-future}

\textit{Concrete next steps:}

\subsection{Video SSL Backbones (V-JEPA)}
\textit{Extend to video-based SSL (V-JEPA 2) that can exploit temporal structure across frames. Hypothesis: temporal prediction pre-training aligns better with the planning task than single-frame SSL.}

\subsection{Closed-Loop Evaluation}
\textit{Evaluate JEPA-Diffusion in closed-loop simulation (e.g., CARLA, nuPlan simulator) to validate open-loop findings.}

\subsection{Driving-Specific Pre-Training}
\textit{Pre-train I-JEPA on driving video datasets (nuScenes, Waymo Open) instead of ImageNet. May further improve cross-city features by learning driving-relevant invariances.}

\subsection{Scaling to More Cities and Datasets}
\textit{Extend city splits to Waymo Open Dataset (San Francisco, Phoenix, etc.) and test cross-dataset transfer.}

\subsection{Multi-Modal JEPA-Diffusion}
\textit{Re-introduce LiDAR via a JEPA-style multi-modal encoder to close the gap with methods like Drive-JEPA while retaining generalization benefits.}
